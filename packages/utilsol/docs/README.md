# @gis-js/vue2ol

@gis-js/vue2ol是一个用于vue框架的javascript库，它封装了openlayers,使创建响应式地图变得更容易

## 安装

```bash
npm install @gis-js/vue2ol ol --save
```

## 注册组件

**全局注册**

```javascript
import Vue from "vue"
import Vue2ol from "@gis-js/vue2ol"

Vue.use(Vue2ol)
```

**按需引入**

```javascript
import {Vue2olMap,Vue2olView,Vue2olFeature} from "@gis-js/vue2ol"
```
